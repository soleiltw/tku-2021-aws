# TKU 2021 AWS Classroom

## Setup ENV before python run
* AWS_ACCESS_KEY_ID=<YOUR_KEY_ID>
* AWS_SECRET_ACCESS_KEY=<YOUR_ACCESS_KEY>
* AWS_DEFAULT_REGION=ap-northeast-1
